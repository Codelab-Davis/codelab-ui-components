## Installation

To integrate the DataTable component into your project, import it and the data type it operates on, Payment, from the components directory:

```tsx
import { DataTable, Payment } from "@/components/ui/data-table";
```

Using the DataTable component is straightforward. Hereâ€™s a basic example:

  {
