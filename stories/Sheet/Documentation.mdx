## Installation

To integrate the Sheet component into your project, import it from the components directory:

```tsx
import { Sheet } from "@/components/ui/sheet";
```

## Usage

Using the Sheet component is straightforward. Hereâ€™s a basic example:

```tsx
import {
    Sheet,
    SheetContent,
    SheetDescription,
    SheetHeader,
    SheetTitle,
    SheetTrigger,
} from "@/components/ui/sheet";

import { Button } from "@/components/ui/button";

<Sheet>
    <SheetTrigger>
        <Button>Open</Button>
    </SheetTrigger>
    <SheetContent>
        <SheetHeader>
            <SheetTitle>Edit Profile</SheetTitle>
            <SheetDescription>
                Make changes to your profile here. Click save when
                you're done.
            </SheetDescription>
        </SheetHeader>
    </SheetContent>
</Sheet>;
```
