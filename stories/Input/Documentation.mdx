## Installation

To integrate any of the Input components into your project, import them from the components directory:

```tsx
import {
  Input,
  LabelInput,
  FileInput,
  ButtonInput,
  FormInput
} from "@/components/ui/input";
```

## Usage

Here is a simple example of using the base 'Input' component:
```tsx
<Input/>
```
This renders the basic input component

<Story of={InputStories.Default}/>

The example above demonstrates a simple input with the default styling. You can further customize it using the available props and variants.

## Props

Depending on the type of input component, there are different props that can be managed. In each component, there
is a control area where you can toy around with the props depending on your needs.

## Best Practices

When using the Input component, consider these guidelines to ensure a consistent and user-friendly experience:

1. **Variant Selection:** Choose the appropriate variant to clearly communicate the input's importance.
2. **Concise Text:** Keep input and button labels short and action-oriented.
