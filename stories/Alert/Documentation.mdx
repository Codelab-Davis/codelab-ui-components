## Installation

To integrate the Alert component into your project, import it from the components directory:

```tsx
import { Alert } from "@/components/ui/alert";
```

## Usage

Using the Alert component is straightforward. Hereâ€™s a basic example:

```tsx
import {
    Alert,
    AlertDescription,
    AlertTitle,
} from "@/components/ui/alert";

import { TriangleAlert } from "lucide-react";
<Alert variant="default">
    <TriangleAlert className="w-4 h-4" />
    <AlertTitle>Warning!</AlertTitle>
    <AlertDescription>
        Changes may not be saved if you leave this page.
    </AlertDescription>
</Alert>;
```

## Best Practices

When using the Alert component, consider these guidelines to ensure a consistent and user-friendly experience:

1. **Concise Text:** Keep Alert titles and descriptions short and urgent.
2. **Icon Usage:** Incorporate icons that convey caution.
